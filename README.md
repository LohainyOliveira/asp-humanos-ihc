# Sistema de Admissão de Emergência - IHC

Projeto desenvolvido para a disciplina de Interação Humano-Computador.

## Aplicação dos 5 Pilares

### 1. Gestalt e Percepção
Uso de fieldsets para agrupamento por proximidade e similaridade.

### 2. Uso de Cores e Contraste
Somente o botão principal possui alto contraste.
Mensagens de erro utilizam cor + texto explicativo.

### 3. Memória e Chunking
CPF e telefone possuem máscara automática para reduzir carga da memória de curto prazo.

### 4. Foco e Atenção
Interface limpa, sem menus ou distrações visuais.

### 5. Redução da Carga Cognitiva
Uso de consistência nos rótulos.
Aplicação de Progressive Disclosure para histórico médico complementar.

## Técnica mais desafiadora

A aplicação de Progressive Disclosure foi a mais desafiadora, pois exigiu equilibrar simplicidade inicial com a possibilidade de expansão sem comprometer a fluidez da tarefa emergencial.
